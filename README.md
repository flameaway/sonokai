<h2>Installation</h2>

If you are using nvim put in your init.vim folder

```
Plug 'https://github.com/flameaway/sonokai'

let g:sonokai_style = 'shusia'
let g:sonokai_better_performance = 1
let g:lightline = {'colorscheme' : 'sonokai'}
''`
